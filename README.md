## 나는 졸업 할 수 있을까 ?



### 개요

초기엔 학번과 비밀번호를 입력하면 학교 포털사이트에서 정보를 크롤링 해 동작하는 프로그램을 만들려고 했지만

UIS 에서 정보를 크롤링 하는 것 자체가 금지되어 있어 나 혼자 사용하는 용도로 아주아주 간단하게 만들어 보려고 한다.

### 기능

1. 학기 별 학점 조회
2. 학기 별 학점 입력
3. 졸업 조건 충족 여부 검사
